Reasonable Footnotes

Reasonable Footnotes is a tiny, accessible, progressively-enhanced script that transforms standard website footnotes into slightly nicer website footnotes.

Footnotes Example

This section contains a handful footnotes. In the original markup, these footnotes are anchor elements that will scroll the browser down to the footnotes section. This behavior has been altered by Reasonable Footnotes and now the footnotes will be displayed in-place.

Here is a footnote that uses an ellipsis … as its link text. Here's another that uses the footnote's id2. This third footnote3 contains lots of content that will be scrollable when opened.

Footnotes can be keyboard focused and closed by pressing the esc key.

Can I use this?

Reasonable Footnotes is free software. The full, annotated source codeis available, and it is released here under the MIT License.

The current version (0.2.0) is usable, but in steady development and not for the faint of heart or the wary of bugs.

Similar Projects

For those in the market for better website footnotes, but who are not convinced Reasonable Footnotes, there are other projects that may be more appealing. This is not a new idea, there are other implementations to choose from, some of them are surely better.

Bigfoot.js

Bigfoot.js is "a jQuery plugin for creating exceptional footnotes."

Littlefoot.js

littlefoot.js is "a lightweight JavaScript library that creates exceptional footnotes."

Hugo Footnotes Popup.js

Hugo Footnotes Popup.js is "Pure JavaScript plugin to make your footnotes displayed in a panel."
